# 仿真与波形（中文）

## 1. 依赖

- **Verilator**、**GNU make**、**g++**  
- **Python 3**（用于生成 `tests/program.hex`）

在 **WSL2 / Ubuntu** 上可一键安装：

```bash
sudo apt-get update
sudo apt-get install -y verilator make g++ python3
```

Windows 本机若无上述工具，推荐使用 **WSL2**，在项目目录下通过 `/mnt/d/...` 访问仓库。

## 2. 生成程序并仿真

```bash
# 示例：矩阵乘内核
python3 apps/app_cuda_matmul.py
cd sim && make clean && make run
```

仿真结束后：

- **波形文件**：`sim/sim_trace.vcd`（Verilator `--trace` 生成）  
- **内存导出**（若顶层触发）：`sim/dump_mem_0_0.txt`

用 **GTKWave** 打开 VCD：

```bash
gtkwave sim/sim_trace.vcd
```

## 3. FP4 / Tensor Core 单独测试与波形

已安装 Verilator 时，可在仓库根目录执行：

```bash
bash scripts/wsl_sim_fp4.sh
```

**Tensor Core（`TCDP4`）**：

```bash
python3 apps/app_tensor_test.py
cd sim && make run
# 或
python3 tests/test_tensor_only.py
```

**仅 FP4**：

```bash
python tests/test_fp4_only.py
```

成功运行后同样生成 `sim/sim_trace.vcd`，可在波形中查看 `gpu_top`、`alu` / `tensor_core` 等信号。

## 4. 本机无法安装 Verilator 时

1. 使用 **WSL2**，在 Ubuntu 内按第 1 节安装依赖后再仿真。  
2. 或使用具备 Verilator 的 **CI / 远程 Linux 环境** 编译运行 `sim/Makefile`。
